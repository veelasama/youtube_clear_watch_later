# YouTube Clear Watch Later

Скрипт для быстрой очистки списка «Посмотреть позже» (Watch Later) в YouTube.

## 📌 Описание
Этот проект помогает автоматически удалить все видео из плейлиста «Посмотреть позже», чтобы не делать это вручную по одному видео.

## 🚀 Инструкция по использованию

1. Откройте YouTube в браузере и перейдите в раздел **«Посмотреть позже»**.
2. Нажмите `F12` или `Ctrl + Shift + I`, чтобы открыть **Инструменты разработчика**.
3. Перейдите на вкладку **Console** (Консоль).
4. Скопируйте код из файла `remover.js` и вставьте его в консоль.
5. Нажмите `Enter` и дождитесь завершения процесса.

## ⚠️ Важно
* Скрипт работает только в браузере.
* Перед запуском убедитесь, что страница полностью прогрузилась.
* Если YouTube обновит интерфейс, селекторы кнопок могут измениться.
